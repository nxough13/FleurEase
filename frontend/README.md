## Installation

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Frontend runs on: http://localhost:5173

## Build for Production

```bash
npm run build
```

## Technologies Used

- React 19.1.1
- Vite 7.1.2
- React Router DOM 7.8.2
- Material-UI 7.3.2
- Axios 1.11.0
- React Toastify 11.0.5
- Recharts 3.3.0
- SweetAlert2 11.26.3
- React Bootstrap 2.10.10
